## Write your own skills

You can also author your own skills by creating a new subfolder and adding a `SKILL.md` file.

Recommended pattern:

- One skill per directory
- A short, concrete goal at the top
- Clear step-by-step instructions
- Explicit constraints (OS, required apps, permissions, etc.)
- Example inputs / expected outputs when helpful
